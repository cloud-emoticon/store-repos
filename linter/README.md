# linter

Linter for store repos

## Prerequisites
* `Node.js`
* `yarn`

## Prepare

```bash
yarn
```

## Build (Optional)

```bash
yarn built-ts
```

## Run

```bash
yarn run-linter
```

## Generate schemas

If you make any changes to [src/api/](src/api/), run this script to generate new JSON schemas for emoticon authors.

```bash
yarn gen-schema
```
